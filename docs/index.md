# Bash & GNU/Linux


## :simple-gnubash: Bash

Bash es la abreviación de *Bourne Again SHell*.
Es una *shell* de Unix, la cual es un macroprocesador que ejecuta comandos y es una de las terminales de uso más popular en Linux.
