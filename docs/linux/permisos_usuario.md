

# Permisos de usuario

## Modificar

- Modificar el archivo `/etc/sudoers`
- Crear un archivo en la carpeta `/etc/sudoers.d/`





## Ejecutar como sudo sin contraseña



```
ALL ALL=(ALL) NOPASSWD: ruta_script
```
